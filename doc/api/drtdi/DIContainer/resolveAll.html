resolveAll<T> method

List<T> resolveAll<T>()

Resolves all registered instances of type T.

Useful when multiple implementations of the same interface are registered. Returns instances in the order they were registered.

Throws a ContainerDisposedException if the container has been disposed.

Implementation

List<T> resolveAll<T>() {
  _checkDisposed();
  return _resolver.resolveAll<T>();
}
